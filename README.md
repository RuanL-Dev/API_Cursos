## API_Cursos
This is an API RestFull developed creating and managment of Courses. It's a CRUD application where "professor" could creates and manages his course while could see other courses available on database.


## Technology 

Here are the technologies used in this project.

* JAVA 17+
* Spring Boot
* Spring Data JPA   
* Docker 
* Docker Compose
* PostgreSQL
* Maven
* REST API 
* Hibernate
* JUnit - Testing Framework
* Mockito - Mocking Framework
* Lombok - Code Generation Library
* JWT - Authentication

## Services Used

* PostgreSQL - Database
* GitHub - Version Control
* GitHub Actions - CI/CD
* Postman - API Testing
* Swagger - API Documentation
* Git - Version Control System

## Java libs used in the project:

* bcrypt
* spring-boot-starter-data-jpa
* spring-boot-starter-security
* spring-boot-starter-validation
* spring-boot-starter-test
